```C
#include <stdio.h>
#include <string.h>

void reverse(char *word);

int main ()
{
	char str[101];
	fgets(str, sizeof(str), stdin);
	str[strcspn(str, "\n")] = '\0';
	
	char *word = strtok(str, " ");
	while(word != NULL) {
		reverse(word);
		printf("%s\n", word);
		word = strtok(NULL, " ");
	}
	return 0;
}

void reverse(char *word){
	int len = strlen(word);
	int left = 0;
	int right = len - 1;
	while(left < right){
		char temp = word[left];
		word[left] = word[right];
		word[right] = temp;
		left ++;
		right --;
	}
}
```

- reverse函数返回值为空，直接改变word的地址进行反转
- strtok 字符串分割函数
  ```C
  char *strtok(char *str, const char *delim);
  ```
  第一个参数 str：
  1. 第一次调用：传入要分割的原字符串（如 "olleh dlrow"）；
  2. 后续调用：传入 NULL（表示 “从上次分割的位置继续”）。
  第二个参数 delim:
  分隔符集合（不是单个字符）：比如 " " 表示 “空格是分隔符”，",. " 表示 “逗号、点、空格都是分隔符”。
  返回值	成功：返回当前分割出的子字符串的指针；
  无更多子串：返回 NULL。
