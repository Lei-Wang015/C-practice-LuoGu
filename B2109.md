```C
#include <stdio.h>
#include <string.h>
#include <ctype.h>
int main()
{
	char str[256];
	scanf("%[^\n]s", str);
	int num = 0;
	for(int i = 0; str[i] != '\0'; i++){
		if(isdigit(str[i])){
			num ++;
		}
	}
	printf("%d\n", num);
	return 0;
}

```
# C Library - <ctype.h>
- int isalnum(int c) 
  This function checks whether the passed character is alphanumeric.
- int isdigit(int c)
  This function checks whether the passed character is decimal digit.
